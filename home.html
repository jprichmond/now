Hello There

  • You have reached the personal page of yours truly, Jason Richmond. Some things I do or have done can be gound in the links below. Peruse and enjoy!
  • Web resumé and text version: Given that I rather despise documenting personal achievements, I thought it appropriate to devote far too much time creating a resumé generator using Python and Javascript that I can update easily. (The actual reason for this project is that I needed something simple I could do using Python and Javascript that isn’t one of my bigger, unfinished projects.) Have a look at the seeking folder in the repo if you would like to see how they’re created.
  • Awesome’s Meet ’n’ Eat: A game of memetic consumption created in p5.js as the finale of a 25k-word introduction to programming for kids ages 10–18 while working for South Bend Code School. The code was written using some of that newish ES6 sugar to make it resemble the C-clones and Java proper (along with the intentional ignoring of the strict equality operator). All the graphics including the faces are written in code — not a .gif or a .png in sight. Check out the creating folder in the repo to see the implementation, or better yet, play with the code to your heart’s content in the p5.js editor.
  • Faces Collide: A simple implementation of collisions in p5.js created during a research phase of the Javascript tutorial. The logic was ripped from a math paper on linear algebra.
  • Moon Phased: A exploration in bezier curves within p5.js to create crescents and eyes turned into the phases of the moon. The logic here came from learning about magic number kappa (not even Kappa).
